TSB Updater version beta 1.2

1. はじめに
この度はTSB Updaterをダウンロードしていただきありがとうございます。
本ツールは配布ワールドThe Sky Blessingのパッチアップデートをサポートするツールです。

2. 使用方法
TSB Updater.exeを起動してください。
1. [...]と書いてあるボタンを押し、The Sky Blessingのワールドフォルダを選択してください。
2. [更新を確認]ボタンを押すと、利用可能なアップデートのチェックが行われます。
3. アップデートがある場合、詳細が表示されるので、[更新]ボタンを押します。
4. メッセージダイアログが表示され、アップデート完了です。

2. 注意事項
本ツールで行えるアップデートは、X, Y, Zバージョン表記の、Zの更新のみとなります。
例1：v1.0.0からv1.0.3 = 本ツールでアップデートが可能です。
例2：v1.0.0からv1.1.0 = 本ツールでアップデートはできません。
例3：v1.0.0からv2.0.0 = 本ツールでアップデートはできません。
詳しくは
https://tsb.scriptarts.jp/ホーム/download
の、「過去のバージョンを遊ばれている方へ」をご覧ください。

また、本ツールでアップデートを実行された時、二次創作系のデータパックを導入されている場合、それらが削除される可能性があります。
